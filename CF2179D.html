Codeforces Round 1071 (Div. 3)

Problem Statement

Find a permutation p of the array [0, 1, ..., 2n - 1] that maximizes the Sacred Sum S(p):

S(p) = Σ 2n-1 i=0 popcount(p0 & p1 & ... & pi)

The Strategy

  • The Prefix AND is non-increasing. A bit flipped to 0 is lost forever.
  • To maximize S(p), we prioritize numbers with the most Trailing 1s (\(\text{ctz}(\sim a)\)).
  • If Trailing 1s are equal, we pick the Minimum Value for lexicographical minimal result.

Optimal Comparator (C++)

sort(ans.begin(), ans.end(), [](int a, int b) {
    // __builtin_ctz(~x) counts trailing 1-bits
    int cntA = __builtin_ctz(~a); 
    int cntB = __builtin_ctz(~b);
    
    if(cntA != cntB) 
        return cntA > cntB; // Most Trailing 1s first
        
    return a < b; // Lexicographical Minimal tie-break
});
